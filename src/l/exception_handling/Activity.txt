Activity:

		Create a program that will ask a number from
	the user. The program should keep on asking user
	for a number if the user provides input that is
	not numeric. Make use of 'exception handling'.
	
	Hint:
		Use try-catch inside of a loop statement.
		
	Sample:
	Enter a number: dek
	Not a number!
	Enter a number: pok
	Not a number!
	Enter a number: 2
	You entered 2!
